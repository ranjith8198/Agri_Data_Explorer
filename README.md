🌾 Indian Agriculture Data Analytics (1960 - 2025)
An end-to-end data pipeline analyzing 60 years of agricultural trends across India. This project integrates Python for cleaning, PostgreSQL for relational data management, and Power BI for high-level visualization.

📁 Folder Contents:
Data: Raw and cleaned datasets.

Scripts_Python: Python scripts for data cleaning and initial EDA.

SQL_Queries: Optimized PostgreSQL queries for growth and yield metrics.

PowerBI_SQL: Final 2-page dashboard (Interactive PDF & .pbix).

PowerBI_EDA: Historical research and exploratory documents.

📊 Key Insights:
Calculated yield growth rate trends across 60 years.

Identified top-performing districts for Rice, Wheat, and Cotton.

Established correlations between land area and production output.
